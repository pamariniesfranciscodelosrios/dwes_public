Chuleta BindParam

CHULETA — PDO: execute(), bindParam() y bindValue()

Qué es PDO

PDO (PHP Data Objects) es una capa que permite conectar PHP con distintas bases de datos (MySQL, SQLite, PostgreSQL…).

Permite usar consultas preparadas, más seguras contra la inyección SQL.

Métodos para pasar valores

Método Qué hace Cuándo se ejecuta el enlace Tipo de enlace Permite definir tipo Ideal para
$stmt->execute($params) Envía valores directamente en la ejecución Al ejecutar Por valor (copia) No Consultas rápidas de una sola vez
$stmt->bindValue(':param', $valor, PDO::PARAM_*) Enlaza un valor concreto En el momento del bindValue Por valor Sí Controlar tipos (INT, NULL, BOOL...)
$stmt->bindParam(':param', $variable, PDO::PARAM_*) Enlaza una variable (referencia) En el momento del execute() Por referencia Sí Ejecutar la misma consulta varias veces cambiando variables

Diferencias clave

Situación execute($params) bindParam() bindValue()
Enlaza valor directamente Sí No Sí
Usa referencia (toma el valor al ejecutar) No Sí No
Repetir consulta con distintos valores Regular Sí Regular
Escribir menos código Sí No Regular
Definir tipo de dato (INT, NULL, BOOL) No Sí Sí
Evita error con variables del bucle Sí No Sí

Ejemplos prácticos

  1. Método rápido (una vez)
$stmt = $pdo->prepare("INSERT INTO alumnos(nombre, edad) VALUES(?, ?)");
$stmt->execute(['Ana', 20]);
  1. Rápido con marcadores:

$stmt = $this->pdo->prepare("INSERT INTO coches (marca, modelo, anio) VALUES (:marca, :modelo, :anio)");
        $stmt->execute([':marca' => $marca, ':modelo' => $modelo, ':anio' => $anio]);
3. Reutilizable en bucle (por referencia)

$stmt = $pdo->prepare("INSERT INTO alumnos(nombre, edad) VALUES(:n, :e)");
$stmt->bindParam(':n', $nombre, PDO::PARAM_STR);
$stmt->bindParam(':e', $edad, PDO::PARAM_INT);

$datos = [
  ['Ana', 20],
  ['Luis', 22],
  ['Sara', 19]
];

foreach ($datos as $d) {
  $nombre = $d[0];
  $edad = $d[1];
  $stmt->execute(); // toma valores actuales
}
  1. Tipado directo (valor fijo)
$stmt = $pdo->prepare("SELECT * FROM alumnos WHERE id = :id");
$stmt->bindValue(':id', 15, PDO::PARAM_INT);
$stmt->execute();

Errores frecuentes

Error Causa Solución
SQLSTATE[HY093]: Invalid parameter number Los nombres no coinciden con los marcadores :param Revisa los nombres en bind y en la consulta SQL
Todas las filas insertan el mismo valor Se usó bindParam dentro de un bucle con la misma variable Usa bindValue o variables distintas
Los números se guardan como texto No se especificó tipo Usa PDO::PARAM_INT

Constantes de tipo comunes

Constante Tipo de dato
PDO::PARAM_STR Texto
PDO::PARAM_INT Entero
PDO::PARAM_BOOL Booleano
PDO::PARAM_NULL Nulo

Consejos finales

  • Usa execute($params) para consultas simples o de una sola ejecución.
  • Usa bindParam() si necesitas ejecutar la misma sentencia varias veces cambiando variables.
  • Usa bindValue() si quieres tipar sin referencias.
  • Utiliza siempre placeholders (?, :nombre) y evita concatenar variables directamente.
