Introducción

Función echo y Saltos de línea

Para crear una nueva línea con echo en PHP (enlace), utiliza el carácter de escape \n dentro de comillas dobles (") o \r\n para una compatibilidad más amplia entre sistemas. Sin embargo, para que el salto de línea se visualice en un navegador, debes usar la etiqueta HTML <br> o la función nl2br().** **

  • Para texto plano (consola):

Usa \n o \r\n para generar un salto de línea en la salida de la consola.

<?php
echo "Esta es la primera línea.\n";
echo "Esta es la segunda línea.\n";
echo "Esta es la tercera línea.\r\n"; // Para mayor compatibilidad

?>
  • Para salida en un navegador web (HTML):

La etiqueta <br> o la función nl2br() son necesarias para que el salto de línea se muestre en la página web.

Usando la etiqueta <br>.

    <?php
    echo "Esta es la primera línea.<br>";
    echo "Esta es la segunda línea.<br>";
    ?>
  • Usando la función nl2br().

Esta función convierte automáticamente todos los saltos de línea (\n) en etiquetas <br>

    <?php
    $textoConSaltos = "Esta es la primera línea.\nEsta es la segunda línea.";
    echo nl2br($textoConSaltos);
    ?>

2. Comentarios

PHP soporta tres tipos de comentarios:

  • Comentario de una línea: Se utiliza // o #.

// Este es un comentario de una línea
# Este también es un comentario de una línea
- Comentario de múltiples líneas: Se utiliza /* ... */.

/*
 Este es un comentario
 de múltiples líneas
*/

3. Tipos de Datos en PHP

Como en todos los lenguajes de programación, en PHP puedes crear variables para almacenar valores.

  1. Las variables en PHP siempre deben comenzar por el signo $ .
  2. Los nombres de las variables deben comenzar por letras o por el carácter _ y pueden contener también números.

Sin embargo, al contrario que en muchos otros lenguajes, en PHP no es necesario declarar una variable ni especificarle un tipo (entero, cadena ,…) concreto. Para empezar a usar una variable, simplemente asígnale un valor utilizando el operador =

Programa1.php (Ruta:dwes/UD2/Entrega1/Programa1.php)

Crea varios bloques con diferentes tipos de datos en tu programa y muéstralos guardando alguna captura.

Dependiendo del valor que se le asigne, a la variable se le aplica un tipo de datos , que puede cambiar si cambia su contenido. Esto es, el tipo de la variable se decide en función del contexto en que se emplee.

PHP soporta varios tipos de datos, entre los más comunes se encuentran:

  • Enteros (int): Números enteros.

$numero = 42;
- Flotantes (float): Números con decimales.

$precio = 19.99;
- Cadenas (string): Secuencias de caracteres.

$saludo = "Hola, mundo!";
- Booleanos (bool): Verdadero (true) o falso (false).

$esVerdad = true;
- Arrays: Conjuntos de valores.

$colores = array("rojo", "verde", "azul");
- Objetos: Instancias de clases.

class Coche {
    public $marca;
    public function __construct($marca) {
        $this->marca = $marca;
    }
}

$miCoche = new Coche("Toyota");
- Nulos (null): Representa una variable sin valor.

$vacio = null;

4. Variables

Las variables en PHP se representan con un $ seguido del nombre de la variable. No requieren declaración previa.

$miVariable = "Hola, mundo!";

4.1. Reglas para los nombres de variables:

  • Deben comenzar con una letra o un guion bajo (_).
  • No pueden empezar con un número.
  • Son sensibles a mayúsculas y minúsculas ($variable y $Variable son diferentes).

5. Ámbito de las Variables

El ámbito determina desde qué partes del script se puede acceder a una variable.

  • Global: Variables definidas fuera de cualquier función tienen un ámbito global y pueden ser accedidas desde cualquier parte del script, excepto desde dentro de funciones (a menos que se use global o $GLOBALS).

$x = 10; // Ámbito global

function miFuncion() {
    global $x; // Hace $x accesible dentro de la función
    echo $x;
}

miFuncion(); // Imprime: 10
- Local: Variables definidas dentro de una función tienen un ámbito local, es decir, solo existen dentro de esa función.

function otraFuncion() {
    $y = 5; // Ámbito local
    echo $y;
}

otraFuncion(); // Imprime: 5
// echo $y; // Error: $y no está definida fuera de la función
- Estático: Variables locales que persisten su valor entre llamadas a la función usando static.

function contador() {
    static $contador = 0; // Persiste su valor entre llamadas
    $contador++;
    echo $contador;
}

contador(); // Imprime: 1
contador(); // Imprime: 2
contador(); // Imprime: 3

Conversiones de tipos posibles y los resultados obtenidos: https://www.php.net/manual/es/language.types.type-juggling.php#language.types.typecasting
