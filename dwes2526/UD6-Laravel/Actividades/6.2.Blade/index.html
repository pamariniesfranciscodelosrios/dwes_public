ðŸ’»6.2 Blade activity:

Step-by-step activity for creating views in Laravel using @yield, @section, @include, partials , and components with slots . This activity will walk you through setting up a basic layout system that makes use of all these Blade features.

As usual: dont forget to personalize and investigate

Screenshots and comments within your repository.

Prerequisites

  • Directory: UD6-Laravel/Sesion2
  • Laravel project set up (laravel new blade)
  • Web server running (php artisan serve)

Objective

Create a page layout about a sport team using:

  • @yield and @section for templating
  • @include for partials
  • Blade components and slots for reusable UI blocks

Step 1: Create Base Layout

File: resources/views/layouts/app.blade.php

Command: php artisan make:view layouts.app

<!DOCTYPE html>
<html>
<head>
    <title>@yield('title', 'My App')</title>
</head>
<body>
    @include('partials.navbar')

    <div class="container">
        @yield('content')
    </div>

    @include('partials.footer')
</body>
</html>
  • @yield('title') defines a section that child views can fill.
  • @yield('content') is where the main content will go.
  • @include inserts partial views like navbar/footer.

Step 2: Create Partials

Navbar Partial

File: resources/views/partials/navbar.blade.php

<nav>
    <a href="/">Home</a>
    <a href="/about">About</a>
</nav>

Footer Partial

File: resources/views/partials/footer.blade.php

<footer>
    <p>Â© {{ date('Y') }} My App</p>
</footer>

Step 3: Create a Blade View that Extends Layout

Command: php artisan make:view home

File: resources/views/home.blade.php

@extends('layouts.app')

@section('title', 'Home Page')

@section('content')
    <h1>Welcome to the Home Page</h1>

    @include('partials.message', ['message' => 'You are logged in.'])

    <x-alert type="success">
        <x-slot name="title">Success!</x-slot>
        You have successfully logged in.
    </x-alert>
@endsection
  • @extends links the layout.
  • @section fills in the @yield parts.
  • @include inserts a partial with variables.
  • <x-alert> is a Blade component using a slot.

Step 4: Create Include Partial with a Variable

Command php artisan make:view partials.message

File: resources/views/partials/message.blade.php

@if(isset($message))
    <div class="message">{{ $message }}</div>
@endif

Step 5: Create a Blade Component with Slots

Run artisan command:

php artisan make:component Alert

File created:

  • Component class: app/View/Components/Alert.php
  • View file: resources/views/components/alert.blade.php

Edit View File: alert.blade.php

<div class="alert alert-{{ $type }}">
    <strong>{{ $title }}</strong>
    <div>{{ $slot }}</div>
</div>
  • $type is a property passed in.
  • $slot is the default slot.
  • {{ $title }} comes from a named slot.

Update Component Class: (optional)

public $type;

public function __construct($type = 'info')
{
    $this->type = $type;
}

Step 6: Route and Controller (Optional)

Route Example in web.php:

    return view('home');
